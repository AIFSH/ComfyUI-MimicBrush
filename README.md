## How to use
```
# in ComfyUI/custom_nodes
git clone https://github.com/AIFSH/ComfyUI-MimicBrush.git
cd ComfyUI-MimicBrush
pip install -r requirements.txt
```
`weights` will be downloaded from modelscope automatically!
